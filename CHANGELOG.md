## [0.1.0] - 2021-12-17

Container-based PHP application framework with docker-compose. It comes ready-to-use in development, test, and
production environments.

### Added

- Dockerfile v3.8
- Ubuntu 20.04 (LTS)
- Supervisor 4.1
- NGINX 2.4
- PHP-FPM 8.1 (Disabled by default)
- Node JS LTS (16)
- YARN 1.22.17
